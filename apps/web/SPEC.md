# 모의고사 웹앱 제품 스펙

## 개요

베이스 수학학원 18-27인치 터치스크린 키오스크용 모의고사 웹앱입니다.
3단계 흐름: 튜토리얼 -> 답안 마킹(OMR) -> 채점/결과

---

## 앱 흐름

### 1단계: 튜토리얼 (`/`)

- **5개 스텝**으로 구성, 좌우 슬라이드 애니메이션으로 전환
- **스텝 1**: 시험 안내 (시험지 일러스트 - CSS/HTML로 간단히 재현)
- **스텝 2**: OMR 개념 소개 (시험지+OMR 일러스트)
- **스텝 3**: 객관식 마킹 인터랙티브 연습 (15번 3번 마킹/해제)
- **스텝 4**: 주관식 입력 인터랙티브 연습 (4번 문제 숫자 입력)
- **스텝 5**: 시간 제한 경고 안내 (빨간색 경고 테마)
- **헤더**: '모의고사 모드' 타이틀 + '홈으로' 버튼 (학생 드롭다운 표시 안 함)
- **하단 버튼**: 이전으로 / 튜토리얼 건너뛰기 / 다음 (마지막 스텝: '시험 화면으로 이동')
- **로고**: 이미지 대신 '베이스 수학학원' 텍스트로 대체

### 학생 정보 입력 (튜토리얼 완료 후)

- **시점**: 튜토리얼 완료 후, 시험 화면 진입 전 폼(모달)으로 입력
- **필드**: 이름, 학교, 학년, 번호, 좌석번호 (모두 필수)
- **학년 입력**: [1학년] [2학년] [3학년] 버튼 선택
- **나머지 입력**: 브라우저/OS 기본 가상 키보드 사용
- **검증**: 모든 필드 필수, 비어있으면 해당 필드에 오류 메시지 표시

### 2단계: 시험 응시 (`/exam`)

- **시작 방식**: 대기 화면 표시 후 학생이 '시험 시작' 버튼을 눌러야 시험 시작
- **시험 시간**: 1분 (데모용)
- **OMR 카드 디자인**: Figma처럼 실제 OMR 용지 느낌으로 구현
  - 좌측: 학생정보 영역 (이미 입력된 상태로 표시)
  - 중앙: 객관식 버블 그리드 (14문항, 5지선다)
  - 우측: 주관식 답안 입력 필드 리스트 (11문항)
- **객관식**: 단일 선택만 허용 (터치로 선택/해제/교체)
- **주관식 키패드**: 숫자(0-9) + 지우기(backspace) + 완료 버튼
  - 특수 키(√, /, -): UI에 표시하되 비활성화(disabled)
  - 입력 제한: 최대 3자리
- **타이머**: 하단 프로그레스 바 + 남은 시간 텍스트 + '문제가 생겼나요?' 버튼
- **타이머 경고**: 남은 시간 30초 이하 시 빨간색 경고 테마로 변경
- **타이머 만료**: 자동으로 답안 제출 후 결과 화면으로 이동
- **종료하기 버튼** (우상단): 확인 모달('시험을 종료하고 답안을 제출할까요?') 후 제출
- **문제발생 버튼**: 간단한 도움말/FAQ 모달 표시
- **별도 제출 버튼**: 없음 (종료하기만 사용)

### 3단계: 채점 결과 (`/result`)

- **제출 완료 화면**: OMR 카드 배경 흐리게 + "제출 완료!" 메시지 + "결과 보기" 버튼
- **스캔 애니메이션**: 3초간 빨간색 스캔 라인 애니메이션, 하단에 "과연 몇 점일까요?" 텍스트
- **결과 화면** (자유 디자인):
  - 상단: 점수 크게 표시 + 정답/오답/미답 통계
  - 중단: 객관식/주관식 구분된 문항별 정오 그리드 카드
  - 하단: '다시 풀기' + '홈으로' 버튼
- **다시 풀기**: 튜토리얼 첫 화면(`/`)으로 이동
- **홈으로**: 튜토리얼 첫 화면(`/`)으로 이동

## 기술 결정사항

| 항목 | 결정 |
|------|------|
| 상태 관리 | Zustand (전역: 학생정보, 답안, 타이머) + TanStack Query (서버 상태) |
| 스타일링 | Tailwind CSS v4 |
| 라우팅 | React Router v6 (createBrowserRouter) |
| 폼 처리 | useState (추가 라이브러리 없음) |

## UI/UX 규칙

- 터치 타겟 최소 44px, 권장 48px+
- 텍스트 최소 16px (text-base), text-sm 사용 금지
- 뷰포트: h-dvh / w-dvw, 외부 스크롤 없음
- hover 의존 인터랙션 금지
- 인접 터치 요소 간 최소 8px 간격
- 타이머 숫자: font-mono tabular-nums
